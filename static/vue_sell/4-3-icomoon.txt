https://icomoon.io
接下来，我们就手把手地教大家，如何做图标字体。
这里我们用到的是一个教IcoMoon的工具，我们先来打开它的网站icomoon.io，这是它的主页。
右上角，有一个教IcoMoonApp的按钮，我们点开它，它会跳转到这个页面：
https://icomoon.io/app/
我们可以看到，这个页面，有IcoMoon提供的许多免费的小图标，不过我们不打算使用它。
我们需要导入自己的SVG图片，我们可以点击左上角的Import Icons按钮，
然后我们选择自己的所有SVG图片，选择这些文件，我们就可以把它们上传到这个IcoMoon的网站中，然后我们点击这些小图标，
选中它们，然后点右下角的这个Generate Font这个按钮，来生成图标字体，
然后我们跳到这个页面以后，我们可以看到，在这个图标下面，有一个Get Code按钮，我们点一下，
然后就会出来一段代码，这就是告诉我们如何使用这个图标字体，在HTML中我们定义了一个span标签，
<span class="icon-close"></span>
class名为icon-thumb_up，然后，这个CSS class
.icon-close:before {
  content: "\e905";
}
HTML Entity: &#xe905;
before伪类，会加上一个content，然后，这样我们就可以使用它了。是不是非常简单？
接下来我们就可以点击右下角这个download按钮，去把这个图标自己下载，
那在下载之前，我们可以点击左上角这个Preferences按钮，点击它弹出了这样一个对话框：
这里有一个Font Name的设置，这里我们把它改成sell-icon，
这个Font Name的作用就是我们生成这个图标字体的名称，然后，还有一些其它的选项、配置，
比如说，这个Class Prefix类名的前缀，还有一些浏览器支持的情况，等等，这些设置，我们这里都使用默认的设置。
如果感兴趣的同学可以去修改它们，然后来看看其它效果，这里我们就不展开了。
那设置好了以后，我们就来点这个download按钮，可以看到，它就会生成这样一个sell-icon.zip包，我们把它下载下来。
下载下来以后，我们去把它解压缩，解压之后，就会生成一个sell-icon的目录，我们来看看这个目录结构
这个目录下面有一些文件夹，有一个fonts，这个fonts就是所有的图标字体文件，还有个demo-files，
然后里面有一些demo相关的js和css，有一个demo.html、readme.txt、style.css等。
我们点开这个Read Me，它让我们去open这个demo.html，去看这个效果，然后我们打开这个demo.html，
可以看到，这个页面，和我们线上的页面是非常相似的，我们去把它审查元素看一下，然后我们定位到具体的图标，
我们可以看到，这个图标字体，其实，这个页面结构，和我们刚才，看到的这个Get Code是一样的，
可以看到，就是定义这样一个span标签，引用这样一个class，那这个<span class="icon-thumb_up">的这个类定义，
它实际上是在style.css里面，那我们接下来看一下这个文件，我们打开style.css，
可以看到，它这里面有一个font-face定义，它所有的资源依赖，都是在这个font-face里面fonts目录下的文件里。
然后，接下来，还有一个以icon-开头的css选择器，也就是下面这些以icon开头的class样式。CSS样式。都是在这里定义的。
然后它指定了一个font-family，为'sell-icon'，然后还有其它一些样式，这里随后还有一些优化字体显示样式的定义。
然后，因此，只要我们在文件中，使用了这些css文件，再去定义一个标签，使用CSS定义的以icon-开头的这些样式，
就可以使用不同的图标字体。然后在页面上就会生成不同的icon。
那稍后，我们就可以把fonts目录下的所有文件，以及这个style.css去拷贝到我们项目中去使用。我们就介绍完这个图标字体的制作了。
那，接下来，就让我们开始，项目目录结构的设计。
我们所有代码都会放在这个src目录下，我么会有一个main.js的入口文件，这个app.vue文件是整个页面的vue实例文件。
也就是把整个页面看成是vue的大组件，src目录下，通常会有我们两个子目录。
